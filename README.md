# EduNexus 学分课堂学生端

一个现代化的学分课堂学生端管理系统，使用 Vue 3 + Element Plus + ECharts 构建。

## 功能特性

### 🏠 首页功能
- 👤 **个性化欢迎** - 显示用户信息和学分认证状态
- 📊 **学习统计** - 可视化展示学习时长和进度数据
- 🎯 **技能分布** - 饼图展示学生技能掌握情况
- ⚡ **快速操作** - 选课、查看进度、项目实训快速入口
- 📅 **学习日历** - 显示学习安排和重要日期
- ✅ **待办事项** - 管理学习任务，支持添加、删除、设置截止日期
- 📢 **通知公告** - 显示最新通知信息
- 📚 **最近课程** - 展示最近学习的课程进度

### 📈 学习进度
- 📊 **进度统计** - 总课程数、完成进度、学分情况
- 📈 **学分趋势** - 图表展示学分获取趋势
- ⏱️ **时长分布** - 学习时间统计可视化
- 📋 **课程管理** - 课程列表、筛选和进度查看
- 📝 **作业管理** - 作业列表和提交记录
- 🎓 **考试管理** - 考试安排和成绩查询

### 📚 课程选择
- 🔍 **课程浏览** - 支持搜索、分类筛选
- 📖 **课程详情** - 查看课程大纲、学分要求
- ✅ **选课管理** - 添加/移除课程，学分统计
- 🏷️ **课程标签** - 按难度、类型、企业分类
- 📊 **选课统计** - 已选课程学分统计

### 💼 项目实训
- 🏢 **企业项目** - 真实企业项目列表
- 🔍 **项目筛选** - 按行业、难度、时长筛选
- 📋 **项目详情** - 项目要求、学习目标、收获技能
- ✍️ **报名管理** - 项目报名申请和状态跟踪
- 📈 **实训进度** - 项目完成情况和评价

### 👤 用户系统
- 🔐 **用户登录** - 安全认证登录
- 📧 **忘记密码** - 密码重置功能
- 👤 **个人信息** - 用户资料管理和更新
- 🎓 **学分认证** - 学分获取认证管理

### 🎨 交互特性
- 📱 **响应式设计** - 支持PC和移动端
- 🌙 **主题切换** - 明暗主题支持（规划中）
- 🎯 **智能提醒** - 截止日期、重要事件提醒
- 📊 **数据可视化** - ECharts 图表展示
- ⚡ **实时更新** - 数据实时同步
- 🎨 **现代化UI** - Element Plus 组件库，优秀的用户体验

## 技术栈

### 前端框架
- **Vue 3** - 渐进式 JavaScript 框架，Composition API
- **Element Plus** - 基于 Vue 3 的企业级组件库
- **Vue Router 4** - 官方路由管理器
- **Pinia** - Vue 3 推荐的状态管理库
- **Sass** - CSS 预处理器，支持变量和混入

### 构建工具
- **Vite** - 下一代前端构建工具，快速热更新
- **Vue DevTools** - Vue 调试工具

### 数据可视化
- **ECharts 5** - 企业级图表库
- **Vue-ECharts** - Vue 3 ECharts 封装

### HTTP 客户端
- **Axios** - 基于 Promise 的 HTTP 客户端

## 功能详情

### 🏠 首页 (Home.vue)
**核心功能模块：**
- **用户欢迎区** - 个性化问候语、学分认证状态标签
- **统计卡片** - 已选课程数、完成进度百分比
- **快速操作** - 选课、学习进度、项目实训的快速入口
- **学习统计图表** - 支持周/月视图的学习时长柱状图
- **技能分布图** - 雷达图展示各技能掌握程度
- **学习日历** - 月历视图显示作业和考试安排
- **待办事项** - 任务管理（添加、删除、设置截止日期）
- **通知公告** - 最新通知时间线展示
- **最近课程** - 最近学习的课程进度卡片

**交互特性：**
- 点击日历日期查看当日事件详情
- 点击待办事项查看任务详情
- 响应式图表和数据更新

### 📈 学习进度 (Progress.vue)
**核心功能模块：**
- **进度概览** - 总学分、完成率、进行中课程数
- **学分趋势图** - 时间轴展示学分获取情况
- **学习时长分布** - 饼图显示各科目学习时间
- **课程列表** - 课程筛选、进度条、状态标签
- **作业管理** - 作业列表、提交状态、截止时间
- **考试安排** - 考试时间、地点、成绩查询

**数据筛选：**
- 按课程状态筛选（全部/进行中/已完成）
- 按时间范围筛选
- 按课程类型分类

### 📚 课程选择 (CourseSelection.vue)
**核心功能模块：**
- **课程展示** - 网格布局展示可选课程
- **搜索功能** - 按课程名称、教师、企业搜索
- **分类筛选** - 按难度、类型、行业分类
- **课程详情** - 大纲、要求、先修课程
- **选课管理** - 一键选课、学分统计
- **已选课程** - 已选课程列表和学分汇总

**筛选选项：**
- 学分要求（1学分/2学分）
- 课程难度（初级/中级/高级）
- 开课企业
- 课程状态（可选/已选）

### 💼 项目实训 (ProjectTraining.vue)
**核心功能模块：**
- **项目展示** - 企业真实项目列表
- **项目筛选** - 行业、难度、项目周期筛选
- **项目详情** - 项目背景、技能要求、预期收获
- **报名系统** - 申请报名、进度跟踪
- **实训记录** - 历史项目和评价

**项目类型：**
- Web开发项目
- 移动应用项目
- 数据分析项目
- 系统集成项目

### 🔐 认证系统
**登录功能 (Login.vue)：**
- 用户名/密码登录
- 记住登录状态
- 登录验证和错误处理

**密码重置 (ResetPassword.vue)：**
- 邮箱验证
- 新密码设置
- 安全验证码

## 开发指南

### 环境要求
- Node.js 16.0+
- npm 或 yarn 包管理器
- 现代浏览器（Chrome 80+, Firefox 75+）

### 本地开发
```bash
# 克隆项目
git clone [项目地址]
cd oneCreditClassStudent

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问应用
http://localhost:3000
```

### 生产构建
```bash
# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

### 项目配置
- **API 配置** - 在 `src/config/api.js` 中配置后端地址
- **路由配置** - 在 `src/router/index.js` 中配置页面路由
- **样式变量** - 在 `src/styles/variables.scss` 中定义主题色彩
- **组件库** - Element Plus 组件按需导入

### 开发规范
- 使用 Vue 3 Composition API
- 遵循 ESLint 代码规范
- 采用 Sass 预处理器编写样式
- 组件和页面分离，保持代码整洁

### API 集成
- 所有 API 调用统一在 `src/api/` 目录下管理
- 支持请求/响应拦截器
- 统一错误处理和用户提示
- 支持请求重试和超时处理

### 部署说明
- 项目构建后生成 `dist` 目录
- 支持静态文件服务器部署
- 支持 Nginx 反向代理
- 支持 Docker 容器化部署
